# Iris Flower Classification using Artificial Neural Network (ANN)

## Overview

This project implements an Artificial Neural Network (ANN) using TensorFlow/Keras to classify Iris flowers into three species based on their physical measurements.

The model is trained on the famous Iris dataset and demonstrates the complete machine learning workflow, including preprocessing, feature scaling, model training, evaluation, prediction, and visualization.

---

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## Project Workflow

1. Load Iris Dataset
2. Split Data into Training and Testing Sets
3. Perform Feature Scaling
4. Build ANN Model
5. Train the Model
6. Evaluate Model Performance
7. Predict Flower Species
8. Visualize Accuracy and Loss

---

## Technologies Used

- Python
- TensorFlow / Keras

---

## Model Performance

The trained model typically achieves over **95% test accuracy** on the Iris dataset.
